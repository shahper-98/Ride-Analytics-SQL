# 🚖 Ride-Hailing Analytics SQL Project
## 📌 Project Overview

This project demonstrates the use of SQL Server to analyze a ride-hailing dataset and generate meaningful business insights. The analysis covers ride performance, revenue trends, payment methods, route popularity, and service performance using SQL queries ranging from basic data exploration to advanced analytical techniques.

The project is designed to showcase practical SQL skills commonly required for Data Analyst, Business Analyst, and SQL Developer roles.
## 🎯 Project Objectives

- Create and manage a relational database using SQL Server.
- Import and validate ride-hailing data from a CSV file.
- Perform exploratory data analysis using SQL.
- Generate business insights using aggregate functions and grouping.
- Analyze revenue, ride trends, and route performance.
- Apply advanced SQL concepts such as Common Table Expressions (CTEs), Window Functions, and Views.
- Develop a portfolio-ready SQL project demonstrating real-world analytical skills.

## 🛠️ Tools & Technologies

- SQL Server
- SQL Server Management Studio (SSMS)
- CSV Dataset
- Git & GitHub

## 📊 Dataset

The dataset contains ride-hailing transaction records with information such as:

- Ride ID
- Date
- Time
- Service Type
- Source
- Destination
- Distance
- Duration
- Total Fare
- Payment Method
- Ride Status
- Miscellaneous Charges

The dataset contains approximately **50,000 ride records** used for analytical reporting and SQL practice.
